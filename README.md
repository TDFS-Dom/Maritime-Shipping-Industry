# Maritime Shipping Industry — Knowledge Base

## Giới thiệu

Bộ tài liệu kiến thức nền tảng về ngành hàng hải và bunkering (tiếp nhiên liệu tàu biển), phục vụ onboarding và tham chiếu nhanh cho team sản phẩm/kỹ thuật đang xây dựng giải pháp trong lĩnh vực này.

## Đối tượng

- Product Manager, Business Analyst, Developer mới tham gia dự án maritime/bunkering
- Bất kỳ ai cần hiểu nhanh domain knowledge trước khi làm việc với hệ thống Ofiniti

## Learning Path

### Full Flow (cho người hoàn toàn mới về maritime)

```
Maritime → Bunkering → Bunkering Operation → Fuel Type → Regulations → MPA & Singapore Standards → Ofiniti
```

### Shortcut Flow (cho người đã có background cơ bản)

```
Bunkering Operation → Fuel Type → Regulations → MPA & Singapore Standards
```

### Thứ tự đọc khuyến nghị

1. **Fuel Types** — Hiểu các loại nhiên liệu, đặc tính, và tiêu chuẩn chất lượng
2. **Regulations & Compliance** — Hiểu khung pháp lý chi phối việc sử dụng nhiên liệu
3. **MPA & Singapore Standards** — Deep dive vào quy trình bunkering tại Singapore (hub lớn nhất thế giới)

## Khái niệm chính (Quick Glossary)

| Thuật ngữ | Giải nghĩa |
|---|---|
| Bunkering | Quá trình tiếp nhiên liệu cho tàu biển |
| MFM | Mass Flow Meter — thiết bị đo lưu lượng nhiên liệu |
| eBDN | Electronic Bunker Delivery Note — phiếu giao nhiên liệu điện tử |
| B2G | Business-to-Government — hệ thống báo cáo số từ doanh nghiệp lên cơ quan quản lý |
| MPA | Maritime and Port Authority of Singapore |
| ECA | Emission Control Area — vùng kiểm soát khí thải |
| VLSFO | Very Low Sulphur Fuel Oil (≤0.5% sulphur) — nhiên liệu phổ biến nhất sau IMO 2020 |
| ISO 8217 | Tiêu chuẩn quốc tế về chất lượng nhiên liệu hàng hải |
| SS 648/600/709 | Bộ tiêu chuẩn Singapore cho bunkering |
